# fight-gGame

This project is a simple text-based adventure game developed. The game involves a player navigating through a maze-like map using keyboard controls to fight enemies, collect items, and ultimately find the exit.

## Quick Start

For detailed instructions on setting up the project, please refer to the "Installation and Setup" section available [here](https://github.com/whe128/fight-game/wiki/Installation-and-setup).

## Game Control

To navigate the player, use the `W`, `A`, `S`, and `D` keys for movement in four directions. The map is composed of various elements, including walls `#`, enemies `E`, treasures `?`, and an exit `@`.

For further details on gameplay mechanics, please visit the [Gameplay](https://github.com/whe128/fight-game/wiki/Project-summary#game-play) section. Additionally, a comprehensive list of implemented features can be found [here](https://github.com/whe128/fight-game/wiki/Project-summary#implemented-features).
